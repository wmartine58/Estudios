
API de github para perfiles: https://api.github.com/users/<username>
Ejemplo: https://api.github.com/users/octocat
Cuando vaya a realizar la llamada con AJAX: https://api.github.com/users/octocat?callback=?

Usuarios de github para el combobox:
octocat
mojombo

antirez
coding-horror
<ingrese-aqui-su-usuario-de-github>

Cree un documento en JSON con esos usuarios de github y carguelos en el combobox utilizando AJAX. Usted mismo defina la estructura del archivo JSON.

Para incluir iconos como los mostrados en la imagen adjunta, puede utilizar el sitio: http://fontawesome.io/icons/
